File Selection Functionality Supporting ABF, FIF, EDF, and BDF Formats

We enhanced the file selection functionality by allowing users to choose from commonly used file formats: ABF, FIF, EDF, and BDF. Users can now upload files in these formats for further analysis using the software.

The user must place the "fieldtrip-master" folder and the "abfload.m" script in the same directory as the NeuroDominanceTracker GUI executable for smooth functionality.

The FieldTrip toolbox used is an open-source software for advanced analysis of MEG, EEG, and invasive electrophysiological data, created by Robert Oostenveld, Pascal Fries, Eric Maris, and Jan-Mathijs Schoffelen. You can find more details about it in their 2011 publication.

For example, a sample EDF file was uploaded and analyzed. The sampling frequency was 250 Hz, and the analysis revealed dominant theta episodes at 150.07 minutes, as shown in Figure 2.
